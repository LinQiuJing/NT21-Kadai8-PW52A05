# NT21-Kadai8-PW52A05
NT課題：IP接続可能確認ツール

手順
合格条件：
〇IPアドレスを1つのINT型変数に入れること
〇サブネットマスクも同様
〇オクテッド毎に演算をしないこと
〇1つのIPアドレスをドット区切りで表記する機能を実装すること
〇サブネットマスクも同様
〇繋がるか繋がらないかを明確にすること。できるなら色で表現すること
〇実際に判定できるように動くプログラムであること
〇言語は問わない
〇コードはエレガントであること

推奨条件：
〇CIDR表記でもサブネットマスク表記でも動くようにすること
この条件は必須ではない

メモ：main.jsの部分はchatGPT使用あり
